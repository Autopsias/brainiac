# raw/ — immutable captured sources

Sources land here **once** and are never edited or deleted. Each carries
`immutable: true` and a `sha256` integrity anchor. A note that needs to change is
a `brain/` note that links back here (via a `source:` wikilink to this id), never an edit to
a raw file. Commit path: VM `draft_capture` → host `write_note` (AGENTS.md §6).
